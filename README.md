# ルール的なやつ  
## ○武器による威圧  
・ナイフや鉄パイプなどの凶器を使ってRPすればPow45以下の人は遠くなら逃げる。5mほどなら命令できる。（PLも45以下だと命令聞かないとダメ。）  
65以下の人はPowロールをする。それ以上の場合は効果なし。  
銃器の場合は65以下でナイフなどの45以下、70以上はペナルティ付きでPowロール。経歴次第ではペナルティなし。  
  
・ナイフなどの刃物と銃器による攻撃がハード成功以上の場合対象は出血する。出血は1ターンにレベル分のダメージを受ける。  
  
・鉄パイプなどの鈍器による攻撃がハード成功以上の場合対象は怯み状態になる。怯みはレベル×20Dexが下がる。Dexが0になるor怯みレベルが3になったときは昏倒状態に変化する。怯みはラウンド終了時1レベル下がる。  
## ○戦闘関連  
・防御について  
盾などによる防御は攻撃を受けた際に回避,応戦の代わりとして使用可能。防御にはSTRロール防御とDEXロール防御が存在する。防御に使うアイテムはダメージを受けた際そのダメージ分耐久が減る（＊アイテムに装甲がある場合は装甲分軽減した後に耐久が減る）。オーバーした際はそのアイテムを使っていた者にオーバー分のダメージが入る。イクストリーム以上の近接攻撃を受ける又はオーバー分のダメージを喰らうと怯み状態1となる。  
  
・STRロール防御  
近接攻撃に対する防御。失敗又は部位狙い攻撃を受けた際はダメージの肩代わりはできるが怯み1が付与される。  
  
・DEXロール防御  
どの攻撃にも対応できる防御。失敗又は部位狙い攻撃を受けた際はダメージを肩代わりできない。  
  
・銃弾の回避  
回避は銃弾を避けられるが行動済みになる。（銃弾自体は避けられないが避けるような動作をして相手に射撃を失敗させる的なニュアンス。）応戦は不可。盾などによる防御は可能。  
  
・回避のイクストリーム/クリティカル  
回避でイクストリームをした場合次の行動にボーナス+1。クリティカルの場合反撃。  
  
・イクストリームで攻撃した場合ダメージ最大ではなくダメージロールにボーナス+1。  
  
・銃器は至近距離の場合確定で当たる。  
  
・戦闘技能と回避系技能のレベル（レギュラーとか）の差が2以上のとき、部位狙いで攻撃が可能になる。ただし急所への攻撃は不可能。部位狙いできない時は胴体にダメージが入る。

## ○マヌーバー関連  
・マヌーバーは近接技能で行えるが、技能の種類によってはできないものもある。（例えば刀剣の技能で人を突き飛ばすなど。）  
  
・片手で使える銃器の場合マヌーバーを使うことができる。（例えば人質取ったりできる）  
  
○マヌーバーでできることの例  
レギュラー（例）  

| パンチ（格闘）   | 1d3+db                 |
| --------- | ---------------------- |
| 突き（投げ）飛ばす | ダメ（ぶつかった場所で変動）+怯み1     |
| 掴みかかる     | 行動にペナルティ+1。離れるにはSTR対抗。 |
| 転ばせる      | 1d3+転倒1                |
  
ハード（例）  

| キック（格闘）    | 1d6+db                                         |
| ---------- | ---------------------------------------------- |
| 組み付く       | 両者STR対抗しかできなくなる。勝利した側は組みつき解除or1d6+dbの攻撃を選択できる。 |
| 地面に叩きつける   | 1d3+転倒1+怯み1                                    |
| 土などを目にぶつける | 1d3+視界不良1                                      |
  
イクストリーム（例）  

| 急所への攻撃（銃器） | ダメージ2倍+出血3 |
| -------------- | ------------------------------------------------------------------- |
| 急所への攻撃（刃物） | ダメージ1.5倍+出血2（切り捨て） |
| 急所への攻撃（格闘or鈍器） | ダメージ1.5倍+怯み2（切り捨て） |
| 締め付け（格闘） | 両者STR対抗しかできなくなる。締め付け側が勝利すると相手に1d6+怯み6。負けると締め付けから脱出される。 |
| 物の奪取or弾き飛ばし | 対象の手に持っている物を弾き飛ばす。片手が空いている場合は奪取が可能。弾き飛ばした物に使用した攻撃分のダメージ（拳なら1d3+db）。 |
  
## ○状態異常一覧  
・出血  
出血は自ターン開始時にレベル分のダメージを受ける。傷1つ治すごとに1レベル下がる。  
医学で治療された場合治療中は出血が止まり成功すれば出血は0になる。失敗するとかかったターン分のダメージを受ける。傷が0のときは出血レベルも0になる。  
・怯み  
怯みはレベル×20Dexが下がる。Dexが0になるor怯みレベルが3になったときは昏倒状態が追加される。怯みはラウンド終了時1レベル下がる。  
・昏倒  
全ての行動が不可能になる。怯みレベルが0の時に解除され、レベルは1が上限。  
・気絶  
HP1以下又はショックロール失敗時になる状態。HPが2以上,気絶から3ターン経過,Conの成功の3つの条件を満たすことで気絶から復活する。  
・重症（システム変更）  
ショックロールをするレベルのダメージを受けるとなる。常に怯み1,視界不良1が付与される。HPが1度完全回復するかつ1日が経過するまで続く。  
・転倒  
次のターン時に移動することができなくなる。  
・視界不良  
行動にペナルティが+1される。ラウンド終了時2レベル下がる。2レベルで視界衰弱に変化する。  
・視界衰弱  
行動にペナルティが+2される。ラウンド終了時に1レベル下がる。2レベルで失明する。  
・失明  
行動にペナルティが+3される。医療機関で治療することができる。  
## ○探索関連  
・目星で得た情報などはDMに送る。（RPをして情報共有することになるから）  
  
・なるべくキャラクター発言で話す（質問とか以外はRPで話す）。例えばして欲しいことを他の人に伝えたり指示したりするときにRPで伝えるなど。  
## ○技能関連  
○回復系  
・応急手当は手当にいるアイテムなどは特にないがどのように治すか説明しないと治せない(例えばナイフで切られた場所を布で止血するなど。)。怪我に対して適切な治療と判断できないときも無効となる。  
  
・医学は治療に長いターンと医療器具が必要になる。代わりに応急手当てでは治せないような怪我も治せる（例えば足の骨折は応急手当ではあまり効果がないが医学なら骨折前と同程度の能力に戻せる。ただし大怪我を消すのではなく同程度の能力に戻しているだけなので骨折自体は治る訳ではない。)。傷を治すのには2ターン,骨折などの大怪我の治療には4ターンかかる。
